# Krinkle: The Dotfiles

## Install

### Read only
```bash
$ git clone git://github.com/Krinkle/dotfiles.git ~/.krinkle.dotfiles
$ ~/.krinkle.dotfiles/init.sh
```

### Read+Write access
```bash
$ git clone git@github.com:Krinkle/dotfiles.git ~/.krinkle.dotfiles
$ ~/.krinkle.dotfiles/init.sh
```
